# 📚 Трекер прочитанных книг

Консольное приложение для учёта прочитанных книг с модульной архитектурой.

## Описание

Приложение позволяет вести учёт прочитанных книг, сохраняя информацию в JSON-файл. Реализована модульная архитектура, имитирующая командную разработку с использованием Git и GitHub.

## Функционал

- ✅ Добавление книг с указанием автора, названия, оценки (1-5) и даты прочтения
- ✅ Просмотр всех добавленных книг
- ✅ Расчёт средней оценки всех книг
- ✅ Статистика по авторам (количество книг и средняя оценка)
- ✅ Удаление книг
- ✅ Автоматическая проверка дубликатов
- ✅ Валидация вводимых данных

## Структура модулей
```book-tracker/
├── main.py # Точка входа, пользовательский интерфейс, меню
├── storage.py # Работа с JSON-файлом (чтение, запись, удаление)
├── stats.py # Расчёт статистики (средняя оценка, статистика по авторам)
├── models.py # Класс Book с валидацией и сериализацией
├── books.json # Файл хранения данных (создаётся автоматически)
├── .gitignore # Игнорируемые файлы
└── README.md # Документация```

## Модули и их ответственность

| Модуль | Назначение |
|--------|------------|
| `main.py` | Точка входа: меню, ввод-вывод, вызов функций из других модулей |
| `storage.py` | Чтение и запись JSON, поиск, удаление, проверка дубликатов |
| `stats.py` | Подсчёт средней оценки, статистика по авторам |
| `models.py` | Класс Book (поля: автор, название, оценка, дата) |
